JavaScript Tutorial

  • JavaScript ဟာ အလွန်အသုံးများတဲ့ Programing Language တစ်ခုဖြစ်လာပါပြီ။ ယခင်က JavaScript ကို Font-end မှာဘဲ အသုံးပြုခဲ့ကြပေးမယ့် ယခုအခါ Server Side မှာရော Desktop Application များအတွက်ပါ အသုံးပြုနိုင်လာပြီ ဖြစ်ပါတယ်။
  • JavaScript ဟာ Web Programing Language တွေထဲက တစ်ခုဖြစ်ပါတယ်။
  • ဒီအပိုင်းမှာတော့ JavaScript ရဲ့ Font-end အပိုင်းများကိုသာ အဓိကထားလေ့လာသွားမှာဖြစ်ပါတယ်။
  • JavaScript ဟာ လွယ်ကူစွာလေ့လာနိုင်တဲ့ Language တစ်ခုပါ။

JavaScript ကို ဘာကြောင့် လေ့လာရတာလဲ (Why?)

JavaScript ဟာ Web Developer တိုင်းလေ့လာရမယ့် Language သုံးခုထဲက တစ်ခုဖြစ်ပါတယ်။ ဒါကြောင့် Weg Develper ဖြစ်လိုသူတိုင်း မဖြစ်မနေ လေ့လာရမယ့် အရာဖြစ်ပါတယ်။

  • HTML က Webpage ရဲ့ ‌ဒေတာအချက်အလက်တွေပြသဖို့ အသုံးပြုပါတယ်။
  • CSS ကိုတော့ Webpage ကို အလှဆင်ဖို့အတွက် အသုံးပြုပါတယ်။
  • JavaScript ကတော့ Webpage Interactive ဖြစ်ဖို့အသုံးပြုရပါတယ်။
