Download လုပ်ပြီး Zip ဖြည့်လိုက်တာနဲ့ ဖိုင်သုံးခုပါ, ပါတယ်။ custom.css , index.html နှင့် MyanmarGantGaw.ttf ဆိုတဲ့ ဖိုင်သုံးခုတွေ့ပါလိမ့်မယ်။

ပြီးသွားပြီဆိုရင်တော့ Visual Studio ကိုဖွင့်ပြီး File >> Open Foler ကို နှိပ်ပြီး MCG-Editor Foler ကိုရွေးပေးလိုက်ပါ။

ပြီးရင် index.html ကိုဖွင့်လိုက်ပါ။ ပြီးရင် အောက်က ပုံအတိုင်းတွေ့ရပါလိမ့်မယ်။

အပေါ်က ပုံတွေ့ပြီဆိုရင်တော့ အောက်ကအတိုင်း New Folder အသစ်တစ်ခုတောင်းလိုက်ပါ။ ဒီ New Folder က ကျွန်တော်တို့ရေး တဲ့ ပိုစ့်တွေကို ရှင်းရှင်းလင်းလင်း ခွဲထားချင်လို့ပါ။ ဒါကြောင့် Folder Name ကို ကိုယ်ရေးမယ့် Category အတိုင်းပေးနိုင်ပါတယ်။ အောက်မှာတော့ HTML Course အနေနဲ့ ပြပေးထားပါတယ်။

ပြီးရင်တော့ အဲဒီ Folder ကို တစ်ချက်နှိပ်ပြီး New File တစ်ခုတောင်းလိုက်ပါ။ ပြီးရင်တော့ ကိုယ့် Tutorial Name အတိုင်း နာမည်ပေးလိုက်ပါ။ အစဉ်အတိုင်း ဖြစ်စေချင်ရင်တော့ 001_ စသည်ဖြင့်နဲ့ စပြီးပေးပါ။ ပြီးရင် အဲဒီ html file ကို ဖွင့်လိုက်ပါ။

ပြီးတဲ့အခါ index.html ထဲက Code တွေအားလုံးကို Copy လုပ်ပြီး ခုဏက ဖွင့်ထားတဲ့ file ထဲမှာ Past လုပ်လိုက်ပါ။ ဒါဆိုရင်တော့ စရေးဖို့ အဆင်သင့်ဖြစ်နေပါပြီ။

အဲဒီမှာမှ ကျွန်တော်တို့ဟာ <body> tage ထဲမှာဘဲ Code တွေရေးပါမယ်။ ကျန်တဲ့ Code တွေနဲ့ CSS ဖိုင်တွေကို ပြင်ဖို့မလိုပါဘူး။

CSS file ကဟာ လက်ရှိရေးနေသမျှအတွက် သုံးနေတဲ့ CSS ဖြစ်ပါတယ်။ ဒါကြောင့် နောက်ပိုင်းမှာ Update Code တွေလာမှ သာ ထပ်ထည့်ရပါမယ်။ Update လုပ်တိုင်းလည်း အားလုံးကို ပြောပြပေးပါမယ်။

CSS Update လုပ်တဲ့အခါမှာ ‌လင့်တစ်ခုပေးပြီး အဲဒီလင့်ထဲမှာရှိတဲ့ Code တွေကို Select All >> Copy လုပ်ပါ။ ပြီးရင်ရင် custom.css ထဲက Code တွေအားလုံးကို ဖျက်၊ Past လုပ်ပြီး Save လုပ်လိုက်ရုံပါဘဲ။
